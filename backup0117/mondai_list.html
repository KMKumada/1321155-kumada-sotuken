問題1-4:
1 分間で 1 % の充電が可能なスマートフォンの充電器があります。
現在のスマートフォンのバッテリーの残量 n % からこの充電器で
100 % まで充電するのに何分かかるかを求めてください。
例)
現在のバッテリー残量: 30 % → 充電にかかる時間: 70 分
現在のバッテリー残量: 15 % → 充電にかかる時間: 85 分
●入力される値
入力は以下のフォーマットで与えられます。
n
n はスマートフォンのバッテリー残量を表します。
入力値最終行の末尾に改行が１つ入ります。
●期待する出力
スマートフォンの充電にかかる時間を出力してください。
最後は改行し、余計な文字、空行を含んではいけません。
●条件
0 ≦ n ≦ 100
n は整数
入力例1
70
出力例1
30
入力例2
25
出力例2
75
